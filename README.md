# StringTokenizer
Создайте метод с именем analyzeTokens, который обрабатывает текст тремя способами:1.Базовая токенизация:Подсчёт общего количества токеновВыявление уникальных токеновПоиск самого частого токена2.Категории токенов (применяются строго в этом порядке приоритета):Numbers — содержит только цифрыWords — содержит только буквыMixed — содержит хотя бы одну букву и хотя бы одну цифру (независимо от любых других присутствующих символов)Special — всё остальное (токены, не являющиеся Numbers, Words или Mixed)3.Статистика токенов:Средняя длина токенаСамый длинный токенСамый короткий токенМетод должен принимать:String (text) для анализаString (delimiter) для токенизацииboolean (caseSensitive) для анализа с учётом регистраФормат возвращаемого значения должен быть структурированным отчётом со всеми результатами.Важно: Категория Mixed соответствует любому токену, который содержит и хотя бы одну букву, и хотя бы одну цифру — даже если он также содержит специальные символы (например,  является Mixed, а не Special). Special используется только когда токен не подпадает под Numbers, Words или Mixed.mixedSpecial@234
